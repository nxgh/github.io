状态管理是一个非常重要的话题，它涉及到了

- 组件内的状态
- 组件之间的通信
- 组件间数据的共享。

|         | 组件内状态                         | 组件间通信                | 第三方状态管理库 |
| :------ | :--------------------------------- | :------------------------ | :--------------- |
| vue3    | `ref`、`reactive`                  | `props`、`provide/inject` | `vuex`、`pinia`  |
| react   | `useState`、`useReducer`、`useRef` | `props`、`context`        | `redux`          |
| angular |                                    |                           |                  |

vue

- 组件内的状态： ref、reactive
- 组件间通信: props、 provide/inject
- 第三方的状态管理库，比如 vuex/pinia

react

- useState、useReducer
- useContext
- 第三方的状态管理库，比如 redux/mobx
